---ABOUT----

Application allows a user to submit their information in the
registration form, provided it meets the following criteria:

--No fields can be blank
--First and Last names cannot contain numbers
--Birthday must be before the current date (today)
--Email address must be valid
--Passwords must contain at least 8 characters, one must be an uppercase letter and one must be a numbers
--Password and Password Confirmation must match
